Module contents¶

automatminer.base Module¶

Base classes, mixins, and other inheritables.

Classes¶

DFMLAdaptor A base class to adapt from an AutoML backend to a sklearn-style fit/predict scheme and add a few extensions for pandas dataframes.
DFTransformer A base class to allow easy transformation in the same way as TransformerMixin and BaseEstimator in sklearn, but for pandas dataframes.
LoggableMixin A mixin class for easy logging (or absence of it).
